# SavePartsToSTEP
Макрос для сохранения всех деталей активной сборки, которые изготавливаются из труб, в формате STEP.

### Зависимости
- SolidWorks 2014+
- Microsoft Scripting Runtime
- Microsoft VBScript Regular Expressions 5.5
